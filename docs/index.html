Natural-language shell command suggester

Ask for the command you mean — not the syntax you forgot.

pls turns plain English into useful shell commands, explains what they do, rates the risk, and can ask before running them. It’s part helper, part translator, part gentle social experiment in being nice to your computer.

What it feels like

`pls` is at its best when you know what you want done but do not feel like reconstructing the exact command syntax from scratch.

Inspect the current directory

Fast low-risk questions for the terminal you are already standing in.

pls show hidden files here
Check system state

Short health checks and service lookups without remembering the exact incantation.

pls check if jellyfin is running
Do careful file work

Batch operations, renames, and file selection with risk shown before anything runs.

pls move all srt files into a subtitles folder
